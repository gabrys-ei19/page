Przykładowe Zdarzenia

  • click
  • double click
  • mouseup i mousedown
  • mouseover i mouseout
  • mouse move
  • select
  • focus
  • blur
  • change

Dodawanie zdarzeń

Zdarzenia w JS możemy dodać na dwa sposoby:

  1. bezpośrenio w pliku .html :

    input id="a" class="on" type="button" value="on" onclick="nazwa_funkcji()"

  2. za pomocą addEventListener w .js :

    // Funkcja dodająca obserwatora zdarzeń dla tabeli t
    function load() {
    var el = document.getElementById("t");
    el.addEventListener("click", modifyText, false);
    }
